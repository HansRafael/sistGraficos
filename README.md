## How to control
* A W S D -> control (translate) X and Y axios.
* N to up
* B to down
* P to Perspective
* O to Ortho

To rotate axios:
* x
* y 
* z

To change projection:
* o -> ortogonal
* f -> perspective frustum
* p -> perspective openGl

Scale is a function inside operations -> glScaled. It will return a matrix
